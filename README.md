# WWWRedirect

This is a Railtie that provides middleware to redirect naked domains
(mydomain.com) to the www version (www.mydomain.com).

## Installation

Add this line to your application's Gemfile:

    gem 'www_redirect'

## Usage

Nothing to do!  Works out of the box.
